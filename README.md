# GKV Desktop Climbing Simulator

Sebuah proyek simulator panjat tebing (climbing simulator) yang dikembangkan untuk memenuhi tugas/proyek mata kuliah Grafika Komputer (GKV). Proyek ini mensimulasikan mekanik memanjat dalam ruang tiga dimensi menggunakan engine Godot.

---

## 🚀 Fitur Utama
*   **Mekanik Panjat Tebing Desktop:** Kontrol simulasi memanjat yang diadaptasi khusus untuk perangkat desktop (PC/Laptop).
*   **Simulasi Fisika:** Pergerakan dan deteksi genggaman (grip) yang responsif saat memanjat dinding.
*   **Lingkungan 3D:** Desain tebing dan lingkungan tiga dimensi interaktif menggunakan fitur bawaan Godot Engine.

---

## 🛠️ Teknologi yang Digunakan
*   **Engine:** [Godot Engine](https://godotengine.org/)
*   **Bahasa Pemrograman:** GDScript
*   **Fokus Utama:** Grafika Komputer, Simulasi Fisika 3D, dan Desain Interaksi Pengguna.
